# Projeto-1-Simulador-de-Lucro-Real
 Calcular o ganho líquido real descontando depreciação, combustível e taxas.

Este projeto foi desenvolvido para calcular a rentabilidade real de motoristas de aplicativo, descontando custos operacionais que muitas vezes são ignorados.

## 🛠 Tecnologias
- **Python 3**
- **Pandas** para manipulação de dados

## 📊 Métricas Calculadas
1. **Custo de Combustível:** Baseado no consumo médio e preço por litro.
2. **Depreciação e Manutenção:** Estimativa de custo por KM rodado.
3. **Lucro Líquido:** O valor real que sobra no bolso.
4. **Lucro por Hora:** Eficiência financeira do tempo online.

*Projeto focado em análise de dados e otimização financeira.*
